# Miroir_Atbash-crypt

This tool crypt your sentence in miroir atbash

for use , you must compile in LINUX / UNIX distribution

1) make re

Exemple for launch:

user $ ./atbash "this sentence is encoded"

result = gsrh hvmgvmxv rh vmxlwvw
